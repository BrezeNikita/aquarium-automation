# Автоматизация работы аквариума

Творческая работа студента группы ИВТ-25-2б Брезе Никиты.

## О проекте

Симуляция автоматизированной системы управления аквариумом, разработанная на C++ с использованием фреймворка Qt.

### Решаемые проблемы

Владельцы аквариумов сталкиваются с тремя типичными проблемами:

1. **Поддержание стабильной среды** — температура колеблется, вода испаряется, фильтр забивается. Человек должен контролировать все показатели, так как неполадки могут навредить рыбам.

2. **Рутинные операции** — кормить рыб нужно каждый день, воду менять раз в неделю. Это отнимает время и требует дисциплины.

3. **Стресс для рыб** — резкое включение света пугает обитателей. Ловля сачком во время уборки вызывает сильный стресс.

### Реализованные функции

- Контроль работы фильтра с оповещением о засоре
- Плавное управление освещением (рассвет и закат)
- Автоматическое и ручное кормление
- Эвакуация рыб в отдельный отсек для безопасной очистки аквариума
- Журнал событий с временными метками

## Архитектура

Проект содержит два основных класса:

- **AquariumModel** — ядро данных. Хранит параметры аквариума: температуру, уровень воды, количество корма, яркость света, скорость потока фильтра. Генерирует сигналы при изменениях. Наследуется от QObject.

- **MainWindow** — слой отображения и управления. Содержит указатель на модель, графическую сцену, элементы управления. Подписывается на сигналы модели. Наследуется от QMainWindow.

Связь между классами — агрегация. Модель не зависит от окна и может существовать самостоятельно.
